KET LUAN QUAN TRONG: V5 va V6 cho CHINH XAC CUNG MOT KET QUA (41.72%),
du V6 da xoa recurrent_dropout. Dieu nay CHUNG MINH recurrent_dropout
KHONG PHAI nguyen nhan khien LSTM that bai. Day la mot van de kien truc
khac, ton tai tu V5 va van con trong V6.

Nguyen nhan co kha nang cao nhat - "Conv1D + L2 + LSTM" combo:
  Kien truc LSTM hien tai:
    Embedding(48) -> SpatialDropout1D(0.3)
    -> Conv1D(64, 5, relu, l2=0.0005) -> MaxPooling1D(4)
    -> LSTM(64) -> Dropout(0.3) -> Dense(4, softmax, l2=0.0005)

  So sanh voi GRU (HOAT DONG TOT, 90.25%):
    Embedding(48) -> SpatialDropout1D(0.3)
    -> Bidirectional(GRU(64)) -> Dropout(0.3)
    -> Dense(32, relu, l2=0.0005) -> Dropout(0.3) -> Dense(4, softmax)

  Khac biet duy nhat: LSTM co them Conv1D(l2=0.0005) + MaxPooling1D(4)
  truoc lop hoi quy, GRU thi khong.

  Gia thuyet: SpatialDropout1D(0.3) ngay sau Embedding lam mat 30%
  kenh dac trung tren TOAN BO sequence, ket hop voi L2 tren Conv1D
  khien Conv1D+ReLU bi "chet" (dead ReLU - tat ca output <= 0).
  MaxPooling1D(4) tren chuoi toan so 0 tra ve toan so 0.
  LSTM nhan input = 0 tai moi timestep -> hoi tu nhanh ve mot fixed
  point co dinh, khong phu thuoc input -> Dense luon predict 1 class.

  Bang chung gian tiep: GRU (khong co Conv1D) HOAT DONG BINH THUONG
  voi cung SpatialDropout1D(0.3) va L2(0.0005) tren Dense.
  CNN (co Conv1D + L2 nhung dung GlobalMaxPooling, khong co LSTM)
  cung HOAT DONG BINH THUONG (90.35%).
  => Van de chi xay ra khi Conv1D+L2 dua input vao LSTM.
